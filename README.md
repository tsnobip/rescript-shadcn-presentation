# ReScript-Shadcn Presentation

Presentation of [Rescript-Shadcn](https://rescript-shadcn.miriad.studio/) made at Rescript Retreat 2026 in Vienna on April 18.

Slides available online [here](https://tsnobip.github.io/rescript-shadcn-presentation).

To start the slide show:
- `yarn`
- `yarn dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

Learn more about Slidev at the [documentation](https://sli.dev/).

# Demo

run in two different terminal tabs
```
yarn demo res:dev
```
and
```
yarn demo dev
```
To run the demo project in your browser.

Then, to add a new rescript-shadcn component, for example ButtonGroup, do:
```
yarn demo dlx shadcn@latest add @rescript-shadcn/ButtonGroup
```
